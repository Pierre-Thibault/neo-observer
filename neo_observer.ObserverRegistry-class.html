add_observer(self, observer, sent_by=None, named=None, method='__call__')

source code 

Add an observer to the registry. There are four types of registration for an observer:

1) For all events (use sent_by=None, named=None);

2) For a specific sender (use sent_by=the_sender, named=None);

3) For a specific name (use sent_by=None, named=the_name);

4) For a specific sender using a specific name (sent_by=the_sender, named=the_name).

Parameters:
  • observer - The observer to add. An object of type IObserver, or any callable that can called with zero or one argument. If the callable can be called with one argument, this argument will be the event (an object of type Event).
  • sent_by - The sender that the observer want to observe. Optional.
  • named - The name of the event the observer want to observe. Must be a string if specified. Optional.
  • method - Method name to call on the observer. If the observer is IObserver, receive_event will be used regardless of this value. Optional.

send_event(self, event_or_sender, name=None, info=None)

source code 

Send an event to all observers registered for the event.

Parameters:
  • event_or_sender - The event to send of type Event or the sender of the event.
  • name - The name of the event if event_or_sender is not an Event.
  • info - Give more information about an event if event_or_sender is not an Event. It is recommended to use a dictionary. Optional.
